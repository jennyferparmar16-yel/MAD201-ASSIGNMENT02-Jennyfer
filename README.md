# MAD201-ASSIGNMENT02-Jennyfer
A Flutter app that lets users explore movies with posters, genres, and details. Includes Home, Favorites, Profile, About, and Contact screens. Users can mark favorites and view info. Built with clean architecture, navigation, and asset management perfect for learning Flutter basics.

Features:
Home Screen: Displays a grid/list of movies with posters and details
Favourites Screen: Shows movies marked as favourites.
Profile Screen: Basic user info and preferences.
About Screen: App overview and purpose.
Contact Screen: simple form or contact information.
Add/Remove Favourites with real-time UI updates.

Learning Objectives:
Understand flutter widget hierarchy
Implement navigation between screens
Manage app state using setState()
Load and display local assets
Build clean,reusable UI components
